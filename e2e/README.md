# GCP Storage: E2E. 

In that directory, you are going to find, how to deploy an architecture using GCP. The objective of that is use the maximum types of storage in GCP. The diagram of the architecture is these:

![Architecture diagram: All GCP](.img/architecture.png)

You will deploy and use the elements on the architecture by the following options: 

- With Terraform and terminal. 
- Without Terraform (UI).

You can find how to use it in the directory `with_tf`, and how to use it in the directory `with_ui`. The links to the README's of the respective directories are: 

- [With Terraform](with_tf/README.md).
- [Without Terraform (UI)](with_ui/README.md).
